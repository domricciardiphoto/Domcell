2024----------------
4/2
  1. New Intro Screen: Official Application Icons. Overview Content is now labeled as Content Designer.
  2. New Key Commands:
    • Ctrl + Delete = Reload
    • F1 = Go to Start Screen
    • F11 = Full Screen
  3. The application will open in the same size it was closed in. This is the first step towards a Save feature.
  4. The Code Cleaner button is now renamed to Plugins & Tools, which contains options for:
    • Store Locator
    • Banner Scheduler
    • Special Character Tool
    • HTML Manifier
    • AI Story Generator
    Note: The frame that holds applications is in the process of being refreshed with an integrated Menu. It's time for a mature coat of paint as the core is eventually done.
  5. Double-clicking on text to edit has been changed to triple-click to allow selecting text again with a double click.
  6. ECP Code import Button: This will put editable code outside the frame. It's useful for when you want to edit existing ecp code to add links, etc., and replace without adding a lot of new code.
  7. Layout Section Revised: The Add Rows section has been modified with a slider so you can add up to 20 rows in a single click.
  8. Rows Section completely overhauled with sliders. Using these dials, you can make full-blown callouts and banners. The sliders allow for 100x more precision.
  9. Bug Fix: Clicking links in PCR Viewer not opening Modal.
  10. Titles of buttons are using fewer words so they don't overlap on smaller screens.
  11. Note: Background Image on Rows Development has begun.
  12. Note: Tools from the Domcell interface will begin migrating over to the web builder so they are all part of the same code platform. As they do, options will disappear from the Getting Started screen.
  13. Note: Web Builder will be broken up into Production and a DEV version if there are any future Large Feature updates to work out any problems before moving it to the Production version. This will be available in the Getting Started Domcell Home screen.
  14. Features:
    • Row Radius
    • Row Padding
    • Row Background Color
    • Body Background Color
    • Font Color
    • Hide on Desktop & Hide on Mobile
  15. Notes: These options are in "Row Options". Ergonomics will be considered in future updates and eventually background image options as requested.
