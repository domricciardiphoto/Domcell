Key Commands

  • Full Screen:

    Press F11 for full-screen mode.
  • Clear and Restart:

    Press CTRL+R to reset everything.
  • Undo Edit:

    Press CTRL+Z to undo.
  • Redo Edit:

    Press CTRL+Y to redo.

Text Commands

  • Bold:

    Press CTRL+B to make selected text bold.
  • Subscript:

    Press CTRL+E for subscript.
  • Superscript:

    Press CTRL+D for superscript.
  • Right Click on text

    Clamp Text / Delete Text / Copy to Clipboard

Mouse Commands

  • Click to Edit:

    Click any element to edit it in the HTML Editor or Explorer
  • Drag to Move:

    Drag elements to reposition them (when in Sort / Reorder mode).
  • Right Click:

    Right-click an element in Explorer for more options.
  • Right Click on Images:

    Right-click an image for more options.

Editing and Layout Buttons

  • Add Row:

    Click "Add Row" (Blue) to insert a new row.
  • Add Component:

    Click "Add Component" (Orange) to insert a new Component.
  • Delete Element:

    Drag "Numbered Component" (Yellow) to a row to insert a new Component.
  • Edit Element:

    Select an element to start editing. (Explorer / HTML Editor)
  • Delete Element:

    Click Trashcan and delete any selected element.

Image Commands

  • Add Image:

    Use the Image Component to add an image.
  • Edit Image:

    Click on the Effects button for editing options.
  • Delete Image:

    Right-click an image or object in the Explorer for option to delete it.
  • Right Click Image:

    Add Link / Remove Link / Delete Image

Explorer Commands

  • Edit:

    Edit the content in the Object or Edit the Copy and styling of a Text Component
  • Duplicate:

    Duplicate the Object
  • Make Active:

    Put a 100% Component in a Selected Row
  • Empty:

    Empty Selected Object
  • Delete:

    Remove Selected Object
